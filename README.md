# Fraud Detection for E-commerce and Bank Transactions

## Project Overview
This project focuses on detecting fraudulent activities in e-commerce and banking transactions using data analysis and machine learning techniques. The goal is to improve fraud detection accuracy while addressing class imbalance and ensuring interpretability.

## Dataset
- Fraud_Data.csv: E-commerce transaction data
- IpAddress_to_Country.csv: IP to country mapping
- creditcard.csv: Bank transaction data

## Interim-1 Scope
This submission covers:
- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering for fraud detection
- Class imbalance analysis
